## 1. Problem Statement

Multiple crates duplicate test helper functions for creating test data. This leads to:
- Inconsistent test data creation across the codebase
- Increased boilerplate in test files
- Maintenance burden when the `Fact` struct changes (requires updates in multiple places)
- No central place for test fixtures
