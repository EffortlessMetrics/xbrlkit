# xbrlkit-cli

Command-line interface for the xbrlkit XBRL toolkit.

## Installation

```bash
cargo run -p xbrlkit-cli -- --help
```

The CLI is still workspace-bound because it loads checked-in profile data from `profiles/`.
It is not published to crates.io yet.

## Usage

```bash
xbrlkit-cli --help
```

See [xbrlkit](https://github.com/EffortlessMetrics/xbrlkit) for more.
